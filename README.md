# Word Games in C – Jumble & Scrabble

## Description
This project is a **C-based word game program** featuring two popular games: **Jumble** and **Scrabble**.  
It demonstrates **string manipulation, logic building, and interactive programming**, offering a fun and challenging experience for users.
The games include **three difficulty levels** — Easy, Medium, and Hard — to progressively test players’ skills.  

## Features
- Two word games: **Jumble** and **Scrabble**  
- Three difficulty levels: **Easy, Medium, Hard**  
- Interactive console-based gameplay  
- Scoring system to track performance  
- Random word selection for replayability  

## Technologies Used
- **C Programming Language**  

## Concepts Applied
- Arrays and strings  
- Loops and conditional statements  
- Functions  
- Logic and algorithm design  

## How to Run
1. Compile the program using a C compiler (e.g., GCC)  
2. Run the executable in a terminal or console  
3. Follow the on-screen instructions to select the game and difficulty level
